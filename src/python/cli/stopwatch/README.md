# Stopwatch

A `python` package/library tutorial of textual

## Installation

```bash
git clone https://github.com/pr4j3sh/stopwatch
cd stopwatch
```

```bash
python -m venv .venv
```

```bash
source .venv/bin/activate
```

```bash
pip install -r requirements.txt
```

## Usage

```bash
python -m stopwatch.main
```
